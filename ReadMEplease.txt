CARA CLONE DAN PUSH PROJECT

CARA CLONE PROJECT
1. Masuk ke github dan cari file/project yang akan di clone
2. Klik tulisan clone or download dan di situ ada link https/ssh
3. Buat folder untuk menyimpan data hasil clone
4. klik kanan folder tersebut dan pilih "Git Bash Here"
5. setelah terbuka lalu ketik "git clone (copy link https/ssh yang ada)" lalu enter
	examples : git clone https://github.com/muhamadikhsann/project-pertama.git
6. tunggu prosesnya
7. kemudian cek folder yang kalian buat sudah ada file nya atau tidak

CARA PUSH PROJECT
1. Login ke dalam akun GitHub kalian
2. Buka New Repository dan isikan nama sesuai project yang akan kalian push
3. isikan opsi yang kalian ingin kan dan klik create repository
4. setelah itu masuk ke file explorer dan pilih project yang akan kalian push
5. klik kanan di project kalian dan pili "Get Bash Here"
6. ketik "git init" lalu enter
7. ketik "git add *" lalu enter
8. ketik "git commit -m "(isi dengan nama project)"
9. ketik "git remote add origin (masukan https/ssh yang ada di new resitory kalian)
10. ketik "git push origin master" lalu enter
11. refresh browser kalian 


*warning : harus terkoneksi ke internet
